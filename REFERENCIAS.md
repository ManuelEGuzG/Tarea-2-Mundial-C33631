# Referencias consultadas

## Documentación oficial

- [Node.js API oficial](https://nodejs.org/api/) — Módulos nativos: `node:sqlite`, `node:fs`, `node:process`, `node:path`
- [Express.js](https://expressjs.com/) — Framework web para Node.js
- [Express API Reference](https://expressjs.com/en/5x/api.html) — Documentación de la API de Express v5
- [Zod](https://zod.dev/) — Librería de validación y parsing de esquemas
- [SQLite (Node.js)](https://nodejs.org/api/sqlite.html) — Módulo nativo de SQLite en Node.js v22+
